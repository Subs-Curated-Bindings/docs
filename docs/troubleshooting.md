# Troubleshooting

If your binds are not working as expected, use this page before opening a support post.

## Start Here

1. Re-check [Software Configuration](general-setup/software-configuration.md)
2. Re-check [Star Citizen Setup](general-setup/star-citizen-setup.md)
3. Confirm you imported `layout_SUB_Clear_Bindings_480_exported.xml` first, then your joystick-specific layout file
4. Confirm Joystick Gremlin profile is active before launching Star Citizen

## Need Help From Support?

If your issue is not covered in those FAQ channels, follow the pinned instructions and open a post in:

- [Enhanced Bindings Support Forum](https://discord.com/channels/303670222097874945/1006954369800998992)

## What to Include in Your Post

- Joystick hardware and firmware version
- Star Citizen patch/build
- Which profile and XML files you loaded
- What you expected vs what happened
- Screenshots of Joystick Gremlin, HIDHide, vJoy, and USB Game Controllers (`joy.cpl`)

Using that format gets faster, more accurate help.
