# Joysticks Overview

Use this page to quickly find your setup and check release status before you start.

!!! info "Priority and scope"
    The most popular joystick setups are always the top priority for updates and fixes.
    
    More niche setups may be incomplete or have small issues, but they still save hours by letting you continue from an existing baseline instead of starting from scratch.
    
    If you use one of those less common configs, contributions and feedback are welcome.
